## コンセプト

「浮遊する色」

奥行きのある暗い空間に、色絵の具を落としたような滲みを持つ色彩が生まれ、ゆっくりと漂いながら奥へ沈んでいく。  
鑑賞者の操作を起点に、花弁を思わせる有機的な形が空間の最前面に立ち上がり、時間とともに輪郭を溶かしながら遠ざかる体験を目指す。

## モチーフ

- 参照画像にある、暗く濃い空間に鮮やかな花が浮かぶ色彩感をモチーフとする
- 黒は「無」ではなく、深度を持つ空間として扱う
- マウス（またはタッチ）が空間表層をなぞると、最前面に花弁状の浮遊体が発生する
- 発生直後から、絵の具が水中で拡散するような滲み・ぼかしを伴う
- 花弁は複数枚が円環状に重なった構造で表現する

## 体験ゴール

- 鑑賞者が操作するたびに、暗く深い空間へ吸い込まれるような没入感を得る
- 花弁状の色彩が「手前で生まれ、滲みながら奥へ沈む」ことで、空間の深度と湿度感を直感的に感じられる
- 思わず何度も画面をなぞり、深淵をのぞきこみたくなる蠱惑的な体験を生む

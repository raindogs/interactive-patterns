## コンセプト

「浮遊する色」

奥行きのある暗い空間に、色絵の具を落としたような滲みを持つ色彩が生まれ、ゆっくりと漂いながら奥へ沈んでいく。  
鑑賞者の操作を起点に、花弁を思わせる有機的な形が空間の最前面に立ち上がり、時間とともに輪郭を溶かしながら遠ざかる体験を目指す。

## モチーフ

- 参照画像にある、暗く濃い空間に鮮やかな花が浮かぶ色彩感をモチーフとする
- 黒は「無」ではなく、深度を持つ空間として扱う
- マウス（またはタッチ）が空間表層をなぞると、最前面に花弁状の浮遊体が発生する
- 発生直後から、絵の具が水中で拡散するような滲み・ぼかしを伴う
- 花弁は複数枚が円環状に重なった構造で表現する

## 体験ゴール

- 鑑賞者が操作するたびに、暗く深い空間へ吸い込まれるような没入感を得る
- 花弁状の色彩が「手前で生まれ、滲みながら奥へ沈む」ことで、空間の深度と湿度感を直感的に感じられる
- 思わず何度も画面をなぞり、深淵をのぞきこみたくなる蠱惑的な体験を生む

## インタラクション仕様

- 開始条件: 初回 `pointermove` または `pointerdown` で描画開始
- 追従ロジック: 入力座標を `target`、描画座標を `stalker` とし `stalker += (target - stalker) * 0.05` で重鈍に追従
- エフェクト発火条件: `stalker` の移動距離が `68px` を超え、かつ `220ms` クールダウン経過時にクラスタを生成
- 消失条件: 各クラスタの寿命は `5.2s`、初期は滲み状態で出現し中盤で軽く合焦、終盤で再拡散しながら奥へ溶解
- 制限値（同時描画数、寿命、密度など）: 同時クラスタ数 `80` 上限、超過時は古いクラスタから削除

## ビジュアル仕様

- 色設計（主要カラー / 参考作品）: `sources` 収録の参照を基準に、オレンジ / エメラルドグリーン / イエローを主軸にする
- レイヤー構成: 背景（左: 赤黒+青み / 右: 明るい霧） / 前景クラスタ（初期は滲み） / 中景クラスタ（軽く合焦） / 後景クラスタ（再拡散・奥行きフェード）
- 合成モード・ブラー・にじみ設定: `lighter` または `screen` を軸に検証し、クラスタ描画時に半透明重ねとソフトエッジで滲みを表現
- 背景処理: 黒を基調とした深度表現

## パフォーマンス目標

- 目標 FPS: Desktop 60fps / Mobile 50fps 以上
- 計測条件（端末 / ブラウザ）: macOS + Chrome 最新版、iOS Safari 最新版、Android Chrome 最新版
- 劣化戦略（低スペック時のフォールバック）: FPS 低下時は生成密度と最大クラスタ数を自動で段階的に縮小

## アクセシビリティ

- `prefers-reduced-motion` 対応: 有効時は追従速度を上げ、生成密度を 50% に制限
- タッチ操作時の代替挙動: `pointermove` の代わりに `pointerdown`/`pointerup` 中心で生成し、意図しない連続発火を抑制
- コントラスト / 可読性の配慮: UI テキストを重ねる場合は白系文字 + 影で可読性を確保

## 受け入れ条件

- [ ] 黒背景上で、入力位置に対して遅延追従する挙動がある
- [ ] 花弁状の滲み表現が前景で発生し、時間経過で奥行き方向へ沈む
- [ ] Desktop/Mobile の双方で入力に反応する

## コンセプト

「分解と再構成を同時に見る」

1つの信号がどの周波数成分で構成されるかを、
時間領域・周波数領域・回転ベクトル（エピサイクル）で同時に観察できる学習用インタラクション。

## モチーフ

- アナログ計測器のようなグリッド感
- 数式理解より先に視覚理解を得られる導線
- 合成波形が再構成される「生成過程」の可視化

## 体験ゴール

- 波形が周波数成分に分解される様子を直感的に理解できる
- 高調波数を増減したときの再構成精度の差を体感できる
- 回転ベクトル連鎖と最終波形の対応関係を観察できる

## インタラクション仕様

- 開始条件: ページロード直後に再生開始（`prefers-reduced-motion` 時は停止状態）
- 追従ロジック:
  - 波形種別・サンプル数変更で DFT を再計算
  - 表示高調波数変更で再構成波形を即時更新
- エフェクト発火条件: 高振幅成分を周波数スペクトルで強調表示
- 消失条件: 不要な高調波は選択解除で再構成から除外
- 制限値（同時描画数、寿命、密度など）:
  - サンプル数: 32 / 64 / 128 / 256
  - 表示高調波: 最大 64

## ビジュアル仕様

- 色設計（主要カラー / 参考作品）: warm paper base + teal/orange accent
- レイヤー構成: 背景グラデーション / 操作パネル / 3種類の可視化パネル

- コントラスト / 可読性の配慮: 暗文字 + 明背景で一定コントラストを確保
